What this reorders in the 8-item plan

  • Fast path (skip extract) is the #1 lever — extract is 1.6–4.5 s and runs on every turn incl. "hi". A chip tap that skips it saves the most of any single change.
  • Async tail (Stage 2) is real and safe — a consistent ~1.0–1.6 s, post-reply. Confirmed deferrable.
  • The compose rewrite (Stage 4-adjacent) is NOT a latency win — ship it for honesty/quality, gated on the persona-family suite, never justified on speed.
  • Next latency question: can the extract LLM be made faster / cached / skipped for the known-intent turns? That is where the ~3 s lives.
